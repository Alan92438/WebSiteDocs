FAQ

How do I convert a chart to image?

Please use RenderTargetBitmap Class.

How do I render a chart that is not in thew screen?

This is not supported for now, keep an eye on #415.

How do I zoom or pan grammatically?

As easy as setting the Axis.MinValue and Axis.MaxValue properties,if you need more information please see Data Pagination article.

Why am I getting ugly animations?

Reference types are easier to track than value types for the library, if in the example you are following the code is plotting a reference type point, like ObservableValue class and you are reproducing it but plotting double values instead, then you might get a different animation.

Why my chart has a poor performance?

Give it a try to LiveCharts.Geared, it is an independent package that boost the performance of the library, LiveCharts started as only good looking charts, performance was never the target, but then as the popularity of the library went up, I Beto Rodríguez was forced to add this package so the library was a full solution, the Geared package is a really solid alternative for high pricing libraries, the reason why I started this library is to avoid those huge prices, give it a try maybe it also helps you.
